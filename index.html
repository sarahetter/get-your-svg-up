What is SVG

  • Scalable Vector Graphic
  • XML
  • A code based graphic format - easier for humans to read
  • images defined by geometric primitives as opposed to bitmaps
  • built-in graphics effects such as clipping and masking operations, background blend modes, and filters.

Why SVG?

  • Scalable!
  • High Fidelity
  • Resolution Independent
  • Small File Size
  • Script-able
  • Animate-able
  • Quick to modify
  • Semantic
  • Accessible
Semantic- in comparison to icon fonts

SVG Coordinate System

  • Viewport
  • viewBox

Gets weird!

Viewport

Viewport is the visible container.

Once the width and height of the outermost SVG element are set, the browser establishes an initial viewport coordinate system and an initial user coordinate system.

If SVG is drawn outside of viewport, it'll just be cropped off.

Viewbox

Starts at 0 0 origin, then goes to 200x and 200y (down and right)

Can be thought of as setting the "units" within the viewport

So if viewbox is same, then 1px=1 unit

If viewbox is smaller, then do math. (viewbox 50, width 200, then 1unit = 4px

If aspect ratios are different, you can do stuff with preserveAspectRatio, but we won't go into that today.

Polygon

Creates a closed polygon

Points are x & y coordinates, separated by space. last coord is attached to first

Path

Commands in d

  • M x y- Move to (beginning of path)
  • L x y - line to x and y
  • H x - horizontal line to y
  • V y - vertical line to y
  • C x1 y1, x2 y2, x y - cubic bezier
  • Q - quadratic bezier
  • A - arc

Can be just lines, or can be closed with Z to create a shape.

Group

Good for moving things (can add x and y), classes for styling, animation, javascripting, etc.

Optimizing SVG Code

  • Remove custom attributes
  • Remove doctype
  • Add semantic classes, if this wasn't done in the graphics program (name of layer is added as a class in illustrator at least)
  • Check shapes, if something is a circle, then it should be a circle tag, not a huge path. Sometimes designers do interesting things.
  • Tools!: http://petercollingridge.appspot.com/svg-optimiser
